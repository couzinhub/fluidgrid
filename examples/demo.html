Fluidgrid Example : The Grids

Test grid for all classes

This document is used to demonstrate the capabilities of the FLUIDGRID grid system. This grid uses 12 columns, with 2% gutters, but fluidgrid can be adapted to any number of columns, for any gutter width. FLUIDGRID is build with SASS, it is a CSS ONLY framework, and does not require javascript or any additionnal script. Unlike most fluid framework, FLUIDGRID allows you to have columns WITHIN columns.
